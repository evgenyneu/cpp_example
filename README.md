# C++ Hello World and build example

This example shows how to write and build a simple C++ program that adds two numbers.


## Compile

```
make
```

## Run


```
./hello
```

## Cleaning

Remove the *.o files:

```
make clean
```

Keep only sources by remove the *.o files, .depend file and the executable:


```
make distclean
```
